
📦 Product Listing App (React)

A production-level React application that fetches product data from an external API and displays it with search and pagination functionality.

🚀 Features:-

🔍 Search functionality

📄 Client-side pagination

🧠 Custom React Hook for data fetching

🏗 Clean and scalable folder structure

🔄 API abstraction layer

⚡ Performance optimization using useMemo

🎨 Responsive and modern UI

📦 Modular and reusable components
-----------------------------------------------------
🛠 Tech Stack:-

React (Functional Components + Hooks)

Axios

CSS (Modern UI styling)

📂 Folder Structure
src/
│
├── components/
├── hooks/
├── services/
├── utils/
└── App.jsx
🔁 Application Flow

Custom hook fetches products from API

Data is stored in state

Search filters products using useMemo

Pagination slices products per page

Reusable ProductCard renders UI

🌐 API Used

Dummy JSON Products API
https://dummyjson.com/products



▶️ Installation
git clone <your-repo-link>
cd project-folder
npm install
npm run dev
